Analysis settings

With the SettingsWidget the analysis settings can be selected which include the result perspective and the type of the demand vector. In case of multi-regional models, also sector locations can be selected via this widget. In this case, also an API configuration needs to be passed into the widget.

In the example below, the settings widget is linked to a hash configuration that serializes the settings into the hash-part of the URL which can be then used by other widgets. 

            
    var settings = useeio.settingsWidget({
        selector: '.useeio-settings-widget',
        withLocations: true,
        webapi: {
            // web API config ...
        },
    });
    var config = useeio.hashConfig();
    config.join(settings);
