Selecting multiple sectors

With the SectorList widget multiple sectors can be selected from a list. In the example below the sector list is linked to a hash configuration from which another widget could get the selected sectors. 

            
    var model = useeio.model({
        endpoint: './api',
        model: 'GAUSEEIO',
        asJsonFiles: true,
    });
    var sectorList = useeio.sectorList({
        model: model,
        selector: '.useeio-sector-list',
    });
    useeio.hashConfig().join(sectorList);
