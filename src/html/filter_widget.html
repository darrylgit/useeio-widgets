This is a simple example that shows how the filter widget could be used with the HashConfigTransmitter. The result perspective can be configured via the perspective parameter. It understands the following values (the values are case insensitive):

  • direct = Supply chain,
  • supply = Supply chain,
  • supply chain = Supply chain,
  • upstream = Point of consumption
  • point of consumption = Point of consumption

The analysis type can be configured with the type or analysis parameter. It understands the following values:

  • consumption = Consumption,
  • production = Production,
